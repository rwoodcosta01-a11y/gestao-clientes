[![NPM](https://img.shields.io/npm/l/react)](https://github.com/rwoodcosta01-a11y/gestao-clientes/blob/main/LICENSE) 

# Sobre o projeto

http://localhost:8000/clientes/list/

 ### Gestao clientes é uma aplicação feita para que empresas consigam ter o fácil controle dos dados de seus funcioários, podendo assim então, editar dados como, salário, nome, idade, bio até mesmo adicionar fotos se for necessário

A aplicação consiste em um mecânicos de ajuda no menuseamento de lista de funcionários para empresas
